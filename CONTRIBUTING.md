# Contributing

## Setup

```shell script
yarn
```


## Run the tests

```shell script
yarn test
```

Coverage needs to be 100%.


## Lint

```shell script
yarn lint
```


## Edit the docs

Docs are generated from [README.md](./README.md) and [DOCS.md](DOCS.md) and from the docstrings in the code. To re-generate the docs run the following script:

```shell script
yarn docs
```
